# Antoine Goze — Portfolio

Personal portfolio site built with [Astro](https://astro.build) and plain CSS. No paid libraries.

## Develop

```bash
cd ~/workspace/personal
npm install
npm run dev
```

Open [http://127.0.0.1:4321](http://127.0.0.1:4321).

## Build

```bash
npm run build
npm run preview
```

## Content sources

Public profile data from LinkedIn and GitHub (`antoinegoze`).
